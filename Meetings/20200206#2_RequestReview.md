## 분석 및 설계에 대한 여러 질문

- DB 저장이 필요할까요?
  - 간단한 데이터면 json 파일 읽기/쓰기로 해도 되고, 다른 url의 rest api 형태의 저장소 GraphQL, redis

- 지도 API 필요할까요?
